project 2
Symptomatic Assistance

The website aims to let the users know from which disease they may be suffering. It takes the symptoms of the user as input values and predict the disease. A list of specialized doctors near them to whom they can consult is also provided. This will be a multiple page website. The index page will contain the information about the website. It will also have a dashboard section in which a-tag buttons will be present to take the user to different pages. The second page will be the ‘All Available Diseases’ page. This page will display all 41 diseases and 133 symptoms in the database The final page will be the ‘Prediction page’ with 4 inputs for the user to put their symptoms. For the prediction we will apply a few machine learning models and pick the best fitted algorithm according to accuracy. This project involves both Machine Learning and Web Development Technologies. To predict the appropriate disease, first, we will clean the data, then pre-process the data, do some analysis, train the model and apply that algorithm which gives the best accuracy for the machine learning model.
